# Home Assistant Add-on: LoRaWAN Server

LoRaWAN Server for Home Assistant.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

You can use this add-on to install LoRaWAN-Server, which is an open-source (MIT licensed) server that implements the LoRaWAN protocol. For more information, please see [lorawan-server].
